Use RobotAdmin generate_shotgun_request_attributes.py to generate "shotgun" request attributes

Use generate_request_attribute_links.py to print a list of MDA URLs to test the generated request attributes

Copy/Paste the URLs into the Chrome Extension called "Open Multiple URLs" to open them all in multiple browser tabs

Optionally use RobotAdmin generate_request_attribute_json_payload_extracts.py to generate request attributes based on JSON payloads

Use delete_request_attributes.py to delete the ones that "come up empty" and leave behind the ones that work
