Set actionName for an XHR action

Agentconfig
Needed agent module configuration:
Support for XmlHttpRequest

The function actionName allows the change of an action name. Normally the name of the current action will get changed.
If you wan't to change a specific (not yet sent) action you can also pass the action id as second parameter.

When you click the button in the example, a new XHR will be triggered.

While this request is active, actionName is called, to rename the action to the string you specified.

Code snippet
//This call will change the name of the xhr action that was created because of the XHR that happened before dtrum.actionName('manually created actionName');
