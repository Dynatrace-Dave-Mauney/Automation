Start an Action That Triggers a Page Change

Agentconfig
Needed agent module configuration:
no extra modules

This example starts an action with enterAction because a setTimeout calls a redirect eight seconds after onload. This is useful when no user input can be associated with an action and automatic user action detection does not work. This can happen when an action is triggered, for example, by setTimeout or setInterval.

Code snippet
dtrum.enterAction('change page', null, null, 'info'); top.location.href = '../index.html';
Example

Page change is done 8 seconds after load via timout!
